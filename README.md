# 🚀 Real-Time Todo Management System

A modern and simple **Task Management Web App** built using **Node.js, Express.js & EJS** 💻

This application helps users manage their daily tasks with a clean dashboard, smooth UI, and proper backend flow ⚡

---

## ✨ Features

📝 Add new tasks with title, description & priority
📋 View all tasks in a structured table
✏️ Edit existing tasks
❌ Delete tasks instantly
🔄 Update task status (Pending → In Progress → Completed)
📊 Dashboard with real-time task summary
🏷️ Status badges for better UI
📱 Fully responsive design

## ⚙️ Installation & Run

```bash
# Clone the repository
git clone https://github.com/your-username/todo-app.git

# Go to project folder
cd todo-app

# Install dependencies
npm install

# Run server
node app.js
```

👉 Open browser:

```
http://localhost:3000
```
